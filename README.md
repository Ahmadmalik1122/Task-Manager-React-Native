# 📝 Smart Task Manager App (MNS UET Multan Project)

## 🚀 Overview

Smart Task Manager is a cross-platform mobile application developed using **React Native** and **Firebase**. The primary goal of this application is to help users organize their daily tasks efficiently while ensuring real-time data synchronization across multiple devices via the cloud.

## ✨ Key Features

- **Firebase Authentication:** A secure login and signup system for personal user accounts.
- **Real-time Cloud Firestore:** Instant data storage and synchronization using Google's NoSQL database.
- **Modern Navigation:** Implemented using **Expo Router** for seamless file-based app transitions.
- **Professional UI/UX:** A clean, responsive, and user-friendly interface designed for a smooth mobile experience.
- **Offline Support:** Basic task persistence to ensure a stable user experience.

## 🛠️ Technical Stack

- **Frontend:** React Native (Expo Framework)
- **Programming Language:** TypeScript
- **Backend/Database:** Google Firebase (Auth & Firestore)
- **Build & Deployment:** EAS (Expo Application Services)
